#　作業開始時
1. git pull
2. /focus-update.mdで作業範囲指定

 # 作業終了後
### 次回タスク不明時はコマンド後に不明の指示

1. /task.done.mdを実行
2. /focus-update.mdを実行
3. /commit-pushを実行

# ルール・ワークフロー変更時
1. `.clinerules/` 配下の該当ファイルを編集・追加する
2. プロジェクトルートで `.\SyncAgentRules.ps1` を実行し、`.agent` および `.AGENTS.md` を同期する
3. 差分を確認し、ルールと自動生成されたファイルを合わせてコミットする
