# プロジェクト概要

## タイトル
フライ！マセイモ・クッキング（仮）

## 一言説明
3キャラ切り替えの「調理コンボ」でマセイモ軍団を倒す、2D横スクロールアクション。WebGL（Unityroom）公開予定。

## コアメカニクス
- **キャラ切り替え:** ケン（切る）/ バーン（焼く）/ アゲミ（揚げる）の3キャラをキー1〜3またはQ/Eで瞬時切り替え
- **調理コンボシステム:** 属性攻撃を順番に当てることで「フライドポテト」「ベイクドポテト」などのコンボが発動
  - 切る→揚げる：フライドポテト（スコア大幅UP＋移動速度アップ）
  - 焼く：ベイクドポテト（体力回復ドロップ）
  - ミスコンボ：消し炭（スコアなし or ペナルティ）
- **ライフ:** 3
- **ステージ:** 3ステージ制、各ステージ末にBOSSマセイモ

## 敵キャラ
- ザコマセイモ：基本の芋
- カタイモ：「切る」を弾く。焼いてから切る必要あり
- ドクドクイモ：倒すと爆発。「揚げる」で無毒化
- BOSSマセイモ：外装を焼いて壊し、中身を切るなどギミックあり

## 操作
- 移動: WASD / 十字キー
- 攻撃: Zキー / クリック（連打でコンボ）
- ジャンプ: スペース
- キャラ切り替え: 1/2/3 または Q/E

## 実装予定機能
- ストーリー＋スキップボタン
- ゲームオーバー画面 / コンティニュー
- エンディング / クレジット
- 一時停止モード
- やり込み：サブクエスト・隠し要素・クリアタイム＆URランキング
