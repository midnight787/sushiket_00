# フライ！マセイモ・クッキング（仮）

> 2D横スクロールアクション — 調理コンボでマセイモを美味しく倒せ！

---

## 概要

性能の異なる3人のキャラクターを切り替えながら、迫りくる芋のマスコット「マセイモ」たちを**調理コンボ**で倒し、最終的にBOSSマセイモの討伐を目指す2D横スクロールアクションゲーム。

- **プラットフォーム:** ブラウザ（[Unityroom](https://unityroom.com/) 公開予定）
- **エンジン:** Unity（2D Platformer Microgame ベース）
- **ステージ数:** 3ステージ
- **プレイ時間目安:** 約10分

## コアメカニクス

### マセイモ・クッキングシステム

3キャラの「調理属性」を組み合わせてマセイモを倒すと、コンボ効果が発動する。

| キャラ | 属性 | 特徴 |
|--------|------|------|
| スライサー・ケン | 切る | 攻撃速度速・リーチ短、コンボ起点 |
| グリル・バーン | 焼く | 高火力・動作遅め、耐久敵に有効 |
| フライヤー・アゲミ | 揚げる | 中距離・広範囲、コンボの締め |

### 調理コンボ例

| コンボ | 手順 | 効果 |
|--------|------|------|
| フライドポテト | 切る → 揚げる | スコア大幅UP + 移動速度UP |
| ベイクドポテト | 焼く | 体力回復アイテムドロップ |
| 消し炭（失敗） | 同属性連打 or 相性NG | スコアなし・ペナルティ |

## セットアップ

### 必要環境

- Unity Hub（推奨：Unity 6000.x 系）
- Git

### クローン手順

```bash
git clone https://github.com/midnight787/sushiket_00.git
cd sushiket_00
```

Unity Hub → Open → クローンしたフォルダを選択して起動。

### ブランチ運用

```
main          # 統合ブランチ（常に動く状態を保つ）
feature/xxx   # 機能開発（例: feature/combo-system）
```

- `main` への直pushは禁止、PRを通すこと
- 作業前に必ず `git pull origin main`

### フォルダ規則（Assets以下）

```
Assets/
├── Characters/     # キャラスプライト・アニメーション（Dsgn A）
├── Backgrounds/    # 背景・タイルセット（Dsgn B）
├── UI/             # HUD・メニュー・エフェクト（Dsgn B）
├── Scripts/        # C#スクリプト（Prgr A/B）
│   ├── Player/
│   ├── Enemy/
│   └── System/
└── Scenes/         # シーンファイル
```
