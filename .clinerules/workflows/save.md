# /save — ローカルにスナップショットを残す（pushしない）

作業の区切りで「とりあえずローカルに保存」したい時用。リモートには送りません。

## 手順
1. 「git status」で変更を確認する。変更が無ければ「変更なし」と報告して終了。
2. 秘密情報が含まれていないか確認する。
3. 次のコマンドを実行:
   - git add -A
   - git commit -m "保存: <現在時刻と簡単な内容>"
4. コミットできたことを日本語で報告する（push はしない）。

## 注意
- このワークフローは push しない。共有したくなったら /commit-push を使う。
- 破壊的操作は一切しない。
