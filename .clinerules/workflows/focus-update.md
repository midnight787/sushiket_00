# /focus-update — 作業フォーカスを activeContext.md に反映する

今日の作業内容・フォーカスが変わったときに activeContext.md を書き直す。

## 手順

1. `cline_docs/activeContext.md` と `cline_docs/progress.md` を読み込む。
2. ユーザーに以下を確認する（まとめて聞いてよい）:
   - 今日（このセッション）で着手・完了したこと
   - 次のセッションで着手する予定のこと
   - 今の懸念・ブロッカーがあれば
3. 現在の `activeContext.md` の以下のセクションを書き直す:
   - `## 今いるフェーズ` — フェーズが変わっていれば更新
   - `## 各担当の現在地` — 関係する担当者の行を更新
   - `## 直近の完了済み作業（技術）` — 今回完了したものを追記
   - `## 現在の懸念・注意事項` — 解消されたものを削除、新規追加
   - `## 次のセッションで着手すること` — 書き直す
   - ファイル冒頭の `**最終更新:**` を今日の日付に変更する
4. 書き直した内容をユーザーに見せて確認を取る。
5. Yesをもらったら `cline_docs/activeContext.md` に書き込む。
6. 完了を日本語で報告する（コミットはしない。`/save` または `/commit-push` を使う）。

## 注意

- 書き直しは **上書き**（追記ではなく全体を新しい状態に）する。
- 古い情報を残さない。迷ったらユーザーに確認する。
- `progress.md` も更新が必要そうなら `/task-done` も合わせて使うよう提案する。
