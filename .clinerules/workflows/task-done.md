# /task-done — タスク完了を progress.md に記録する

完了したタスクのチェックボックスを更新し、日付を記録する。

## 手順

1. `cline_docs/progress.md` を読み込む。
2. ユーザーが「完了した」と伝えたタスクを特定する。
   - 曖昧な場合は「どのタスクですか？」と確認する。
3. 該当行の `- [ ]` を `- [x]` に変更し、行末に `（YYYY-MM-DD完了）` を追加する。
   - 例: `- [x] ケン待機・走りアニメ（2026-05-28完了）`
4. 「進行中 🔄」にあるタスクなら「完了済み ✅」セクションに移動する。
5. 変更内容をユーザーに見せて確認を取る。
6. Yesをもらったら `cline_docs/progress.md` に書き込む。
7. 完了を日本語で報告する（コミットはしない。`/save` または `/commit-push` を使う）。

## 注意

- 複数タスクまとめて完了した場合はまとめて更新してよい。
- 「既知の課題 ⚠️」セクションも解消されていれば同時に削除する。
- 書き込み前に必ずユーザーの確認を取る。
